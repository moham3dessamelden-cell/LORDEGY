
# LORD STORE Bot V3 - Market Style + Stock Reservation

النسخة دي معدلة لتكون أقرب للشكل اللي في الصور:

## الجديد
- واجهة منتجات على شكل أقسام مثل ChatGPT / Gemini / Adobe / Outlook.
- كل قسم يظهر بعدد المنتجات.
- داخل القسم كل منتج يظهر بالمخزون: `Product (9)`.
- تفاصيل المنتج بشكل احترافي:
  - Delivery
  - Price
  - Stock
  - Duration
  - Warranty
- تأكيد شراء قبل الخصم.
- لو الرصيد غير كافي يظهر زر `Deposit Now`.
- لو المنتج مخلص وفيه `allow_preorder` يظهر زر:
  `Pay & reserve until restock`
- لما العميل يدفع ويحجز والستوك = 0:
  - يتخصم الرصيد.
  - الطلب يبقى `waiting_stock`.
  - أول ما تضيف ستوك لنفس المنتج، البوت يسلّم تلقائيًا للطلبات المحجوزة حسب الأقدم.

## تشغيل البوت
1. انسخ `.env.example` باسم `.env`.
2. ضع `BOT_TOKEN`.
3. غيّر `ADMIN_WEB_PASSWORD`.
4. شغل:
```text
run.bat
```

## تشغيل الداشبورد
```text
run_dashboard.bat
```

وافتح:
```text
http://127.0.0.1:8000/admin
```

## أوامر الأدمن داخل تليجرام

### عرض لوحة الأدمن
```text
/admin
```

### إضافة قسم
```text
/addcategory emoji|name_ar|name_en
```

مثال:
```text
/addcategory 🤖|ChatGPT|ChatGPT
```

### عرض المنتجات
```text
/products_admin
```

### إضافة منتج أوتوماتيك
```text
/addproduct_auto category_id|الاسم|السعر|المدة|الضمان|طريقة التسليم|الوصف|preorder yes/no
```

مثال:
```text
/addproduct_auto 1|ChatGPT Plus 1 Month|355|شهر|25 يوم|Auto ⚡ Instant delivery|Ready subscription service|yes
```

### إضافة ستوك
```text
/addstock PRODUCT_ID
```

ثم أرسل المخزون، كل عنصر في سطر:
```text
CODE-111
CODE-222
CODE-333
```

أي طلبات محجوزة `waiting_stock` لنفس المنتج سيتم تسليمها تلقائيًا بعد إضافة الستوك.

## ملاحظات
- ألوان أزرار تليجرام نفسها لا يمكن التحكم فيها برمجيًا، لكن النسخة تقلد نفس التنظيم والشكل بالـ emojis والصفوف.
- استخدم البوت فقط لمنتجات وخدمات رقمية مسموح لك ببيعها.

- تم إضافة زر الطلبات وزر العملة في القائمة الرئيسية.

# تحديث V4 - تعديل المنتجات من الداشبورد

تم تفعيل صفحة تعديل كاملة لكل منتج داخل قسم المنتجات.

## من الداشبورد
افتح:
```text
http://127.0.0.1:8000/admin/products
```

هتلاقي زر جديد بجانب كل منتج:
```text
تعديل
```

## تقدر تعدل:
- القسم.
- حالة المنتج: نشط / متوقف.
- اسم المنتج عربي.
- اسم المنتج إنجليزي.
- السعر بالجنيه.
- نوع التسليم: Auto من المخزون / Manual من الأدمن.
- المدة.
- الضمان.
- طريقة التسليم.
- السماح بالحجز عند نفاد المخزون Pay & Reserve.
- المخزون اليدوي للمنتجات Manual.
- الوصف عربي.
- الوصف إنجليزي.
- رسالة التسليم اليدوي.

## تعديل المخزون
داخل صفحة تعديل المنتج تقدر:
- تضيف Auto Stock جديد.
- تستبدل المخزون غير المباع.
- تحذف كل المخزون غير المباع.
- تشوف أول 50 عنصر غير مباع.
- عند إضافة ستوك جديد، أي طلبات waiting_stock لنفس المنتج يتم تسليمها تلقائيًا بالأقدمية.
